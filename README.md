# 🚀 IR NETLIFY Scanner

اسکنر سریع و چند‌ترد برای بررسی IP و دامنه‌های مرتبط با Netlify و سرویس‌های Kubernetes ⚡

---

## ✨ ویژگی‌ها

* ⚙️ اسکن همزمان با `ThreadPoolExecutor`
* 🌐 بررسی وضعیت دامنه‌ها با `curl`
* 📊 نمایش زنده Progress و تعداد نتایج پیدا شده
* 💾 ذخیره نتایج در فایل `scan_results.txt`
* 🎨 رابط کاربری رنگی داخل ترمینال
* 🖥️ پشتیبانی از ویندوز و لینوکس

---

## 📦 پیش‌نیازها

* Python 3
* نصب بودن `curl`
